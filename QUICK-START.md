## STEP 1: GitHub (2 min)

1. Go to [github.com](https://github.com)
2. Click **Sign Up**
3. Enter email, password, username
4. Verify email
5. Done! ✅

---

## STEP 2: Create Repository (3 min)

1. Click **+** (top right) → **New repository**
2. Name: `syre-diamond-autos`
3. Description: `Syre Diamond Autos - Luxury Car Dealership Website`
4. Keep it **Public**
5. **DO NOT** add README or .gitignore
6. Click **Create repository**

---

## STEP 3: Upload Files (5 min)

### Easy Method (Web Upload):

1. On your repo page, click **uploading an existing file**
2. Open folder: `/home/moltbot/.openclaw/workspace/syre-diamond-autos-static/`
3. Select **ALL files** (Ctrl+A or Cmd+A)
4. Drag into GitHub upload area
5. Wait for upload
6. Commit message: `Initial launch`
7. Click **Commit changes**

Done! Your code is on GitHub! ✅

---

## STEP 4: Netlify (3 min)

1. Go to [netlify.com](https://netlify.com)
2. Click **Sign Up** → Use **GitHub**
3. Authorize Netlify
4. After login: **Add new site** → **Import an existing project**
5. Select **GitHub**
6. Find `syre-diamond-autos`
7. Click **Deploy site**

Your site is building... wait 1-2 minutes...

**🎉 IT'S LIVE!** You'll get a URL like:
```
https://syre-diamond-autos.netlify.app
```

---

## STEP 5: Formspree (3 min)

1. Go to [formspree.io](https://formspree.io)
2. Sign up with your email
3. Click **+ New Form**
4. Name: `Contact Form`
5. Your email: (where you want to receive enquiries)
6. Click **Create Form**
7. Copy your Form ID (e.g., `xvndbzqo`)

### Update Your Site:

1. Open `index.html`
2. Find: `YOUR_FORMSPREE_ID` (2 places)
3. Replace with your actual ID
4. Save
5. Upload changed file to GitHub (same as Step 3)
6. Netlify auto-updates in 1 minute

## 🆘 Quick Troubleshooting

**Site shows 404:**
- Make sure `index.html` is in the root (not inside a folder)

**Forms not working:**
- Check Formspree ID is correct
- Check you verified your email with Formspree

**Logo not showing:**
- Make sure `images/logo.jpg` exists
- Check file name is exactly `logo.jpg` (lowercase)

**Vehicles not loading:**
- Check `js/vehicles.js` has valid JavaScript
- Look for missing commas or quotes
